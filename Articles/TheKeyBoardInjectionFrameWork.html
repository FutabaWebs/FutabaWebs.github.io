It’s a THREAT 🕸🕸🎃🕸🕸

For starters, this thing is an overall threat in my opinion to be honest from a security standpoint.

Much like how hackers were able to exploit ChatGPT prompts to poison it, the same can be done for it, EXCEPT, ON A MORE DEEPER level if you get crafty enough with it. You can even mesh it with stuff like Windows Recall and other techniques to do some real damage with it if you wanted to!

“BUT FUTABA! IT’S ONLY IN BETA! THEY HAVE TIME TO FIX-”

OK! Yeah, So did ChatGPT and it’s other variants, but like ALWAYS, security is almost NEVER in mind for projects like this!

It’s more prone to security breaches than an actual person and further subject to compromise since it is unable to apply basic operational training procedures in a flexible manner to prevent exploits from connecting.

It’s Rubber Ducky 🕸🕸🎃🕸🕸

However grins menacingly

It's also a rubber ducky that you can now seamlessly make and train for exploits since you can pool all the data from the API and seamlessly tailor it to whatever you need it to do. As someone that WRITES rubber duckies, this thing is a life saver haha. With the API you can even combine some analog stuff with it to give it a boost and have it do whatever you want.

It's even more scary since it could have access to higher levels of privileges you can easily exploit with it should you breach a system running this thing.




Do We NEED This? 🕸🕸🎃🕸🕸

Tbh I don't see people really asking for this product. Seems like they're just trying to hop on the new bandwagon for AI products which is the new trend nowadays. It's not even AI tbh and is just a scripted engine that might require human interaction in the long run, making it non-autonomous which defeats the purpose of it being an AI.

This product, although it sounds nice on paper, has a major flaw in that it might not even be compatible and able to autonomously control various applications that exist in an infrastructure.

Much like with actual keyboard injection, it has to factor various versions and builds of OS and applications as well as the various infinite scenarios that exist while controlling the system. It's simply not possible to do all of that which will again defeat the purpose of it being an AI system since it will require you to know how to program said scenarios for how you intend to use it.
